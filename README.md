# 📚 API de Libros

API REST para gestión de libros desarrollada en Java con Spring Boot.

## 🛠️ Tecnologías
- Java
- Spring Boot
- Maven
- API REST

## 📖 Funcionalidades
- CRUD completo de libros
- Búsqueda por título y autor
- Paginación de resultados

## 🚀 Uso
```bash
# Ejecutar la aplicación
./mvnw spring-boot:run

# Endpoints principales
GET /api/libros
GET /api/libros/{id}
POST /api/libros
PUT /api/libros/{id}
DELETE /api/libros/{id}
```
